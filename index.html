Privacidade Total

Seus dados são salvos somente neste dispositivo. Não há nuvem. Se limpar o cache, os dados somem. Faça backups!

Funcionalidades Principais

  • Olho (): Oculta saldos.
  • Sincronizar Dados (): Força uma nova busca de todos os dados (preços e histórico) da API.
  • Simulador (): Simula o valor do seu portfólio com base em um preço futuro do Bitcoin.
  • Console (): Exibe logs e o status das operações em segundo plano.
  • Ajustes (): Permite carregar backups, bases de dados offline, configurar a API e mais.

Bugs Comuns (API Gratuita)

  • Lentidão: Com muitas moedas no portfólio, a API gratuita pode ficar lenta.
  • Atraso nas Atualizações: A API gratuita pode ter atrasos na atualização dos preços e dados históricos.

Importação via XPUB e Endereços Públicos

  • O app permite importar carteiras Bitcoin usando endereços públicos (P2PKH, P2SH, Bech32, Taproot) ou chaves públicas estendidas (XPUB/YPUB/ZPUB/TR(x)PUB) de carteiras HD.
  • Importamos apenas dados públicos da blockchain. Nunca sua seed, chaves privadas ou informações sensíveis. A importação é somente leitura; você não pode gastar, mover ou assinar transações.

  • Aviso de Privacidade XPUB

    XPUBs revelam TODOS os endereços derivados e todo o histórico de transações da carteira. Serviços externos que consultamos verão seu XPUB. NÃO COMPARTILHE XPUB PUBLICAMENTE.

  • Atualmente o app usa a API gratuita do WalletExplorer para consultar endereços e transações. APIs gratuitas podem ter limites de requisições e atualizações mais lentas.
  • O app não envia dados para servidores próprios, apenas consulta APIs externas. Seu Bitcoin continua seguro na carteira de origem — o app é apenas um visualizador.

Antes de Importar sua Carteira

Importação Somente Leitura

Este app não pede seed nem chaves privadas. Não temos acesso aos seus fundos. A importação serve apenas para ler dados públicos da blockchain (saldo e transações).

Serão importados seus endereços públicos individuais ou chaves públicas estendidas (XPUB/YPUB/ZPUB/TR(x)PUB). A partir disso, o app lerá seu saldo e transações automaticamente.

Alertas de Privacidade Importantes

  • Um XPUB expõe TODOS os endereços derivados da sua carteira.
  • Com um XPUB, todo o histórico de transações da carteira se torna visível.
  • APIs externas que consultarmos verão o XPUB.
  • VOCÊ NÃO DEVE COMPARTILHAR SEU XPUB PUBLICAMENTE.

Atualmente o app usa a API gratuita do WalletExplorer para consultar transações e endereços derivados. APIs gratuitas podem ter limites de requisição e atualização mais lenta.

Sua Segurança é Prioridade

O app não guarda nenhum dado pela internet; tudo fica local no seu dispositivo. Seu Bitcoin continua seguro na carteira original. Esta importação NÃO permite enviar BTC, assinar transações ou alterar a carteira.
